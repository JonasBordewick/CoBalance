# Comparison

The comparison view displays the parameter values of multiple entities side by side — both as a chart and as a table.

---

## Structure

The view has two columns:

- **Left** — list of all entities from the loaded balance snapshot
- **Right** — chart and parameter table for the selected entities

---

## Selecting Entities

One or more entities can be selected in the left table (`Ctrl` or `Shift` click). The right side updates automatically.

The search field in the top-left filters the entity list in real time.

---

## Chart

The chart visualizes the parameter values of the selected entities. Two chart types can be toggled using the switches in the top-right:

**Bar Chart** — grouped bar comparison of all parameters per entity. Well suited for direct value comparison of individual parameters.

**Radar Chart** — spider web diagram that shows the overall profile of an entity at a glance. Good for assessing the balance between multiple parameters.

---

## Parameter Table

Below the chart, a table shows the concrete parameter values of all selected entities side by side. Each entity gets its own column.
