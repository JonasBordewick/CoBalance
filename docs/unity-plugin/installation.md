# Installation

Das **Balancing Framework** kann über den Unity Package Manager direkt per Git-URL in ein bestehendes Unity-Projekt eingebunden werden.
Dadurch ist keine manuelle Installation oder das Kopieren von Dateien in das Projekt erforderlich.

## Plugin über Git-URL installieren

Öffne in Unity den **Package Manager** über:

`Window > Package Manager`

Klicke anschließend auf das **Plus-Symbol** oben links und wähle:

`Add package from git URL...`

Füge dort die Git-URL des Plugins ein und bestätige die Eingabe.

## Installation überprüfen

Nach der erfolgreichen Installation sollte ein neuer Menüeintrag für das **Balancing Framework** in der Unity-Menüleiste erscheinen.
Über diesen Menüeintrag kann das Hauptfenster des Plugins geöffnet werden.
